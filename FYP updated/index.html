HOW TO USE

Select the correct output, given input for each type of Multiplexers

  • 3 to 6 of the Multiplexers input are randomly generated
  • Each input is represented by a letter
  • By default, every letter is a logic 1 input.
  • The operation NOT is represented by `
  • Select the options by clicking on 1 out of the 4 aqua buttons
  • Correct answers will be highlighted in GREEN
  • Wrong answers will be highlighted in RED
  • You only got one attempt per Question
  • "CHANGE QUESTION" will generate a new question
